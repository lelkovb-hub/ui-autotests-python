# Баг-репорт №1

## Заголовок
Промокод GIVEMEHALYAVA не даёт скидку 10%

## Приоритет
Высокий (акция не работает, пользователи не получают обещанную скидку)

## Окружение
- Сайт: https://pizzeria.skillbox.cc/
- Браузер: Chrome 147.0.7727.55
- Дата: 2026-04-10

## Шаги воспроизведения
1. Открыть главную страницу
2. Добавить любой товар в корзину (например, пиццу за 115₽)
3. Перейти в корзину
4. Ввести промокод GIVEMEHALYAVA в поле "Введите код купона"
5. Нажать кнопку "ПРИМЕНИТЬ КУПОН"

## Ожидаемый результат
Сумма заказа уменьшается на 10%. При исходной сумме 115₽ должно стать 103.5₽.

## Фактический результат
Сумма заказа не изменилась (осталась 115₽). Промокод не применился.

## Дополнительная информация
Автотест также не смог применить промокод. Возможные причины:
- Промокод не активен
- Промокод требует минимальной суммы заказа
- Ошибка в логике применения купона
